## 🛎️ 핵심기능
### Chat
- django-channels 라이브러리 이용하여 웹소켓 통신 -> 실시간 채팅 구현
- 챗봇 api 연결을 통한 1대1 챗봇 채팅방 구현
- redis memory 활용
  
### Location
- 위치기반 서비스를 통한 유저 지역 인증기능 (kakao map api)
  
### Trade
- 게시글 CRUD
- 더보기 기능
- 검색기능
- 각 게시물의 채팅, 조회 수 확인가능

### Review
- 채팅방에서 판매자가 거래완료 버튼으로 전환 시 해당 상품 구매자에게만 리뷰 바로가기 버튼 및 리뷰작성기능 활성화

### Mypage
- 유저의 닉네임, 프로필 사진등을 변경가능
- 매너온도 확인가능
- 해당 유저의 판매중, 판매완료, 거래후기등을 한눈에 볼 수 있도록 구현
